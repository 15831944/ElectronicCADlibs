Распаковать на любой диск. ( У меня в примере на диск D:)
Запустить MC11 (MC11xxxx).exe. Закрыть программу. Подключить дополнительные элементы отредактировав файл mcap.dat.
В файле mcap.dat  для подключения дополнительных компонентов добавьте строки, чтобы получилось вот так:
P.S. Находятся ближе к концу файла.

[Component Files]
COMPONENT_FILE=D:\MC11\Standard.cmp
COMPONENT_FILE=D:\MC11\Rus.cmp
COMPONENT_FILE=D:\MC11\Standard_m.cmp

[Shape Files]
SHAPE_FILE=D:\MC11\standard.shp
SHAPE_FILE=D:\MC11\Rus.shp
SHAPE_FILE=D:\MC11\standard_m.shp

[Package Files]
PACKAGE_FILE=D:\MC11\standard.pkg

Можно этого не делать, тогда программа возьмет только основной набор по умолчанию.
Пути к файлам указывайте свои, в папке с программой.
Наборы могут изменяться в зависимости от имени подключаемых файлов элементов.
Подправлена графика пассивных элементов для удобства.

   Запустите программу снова. Элементы должны подключиться.
   Если все правильно, то сохраните в резерв файл mcap.dat с настройками программы и Вашими настройками. Им можно воспользоваться для восстановления настроек в данной папке.
Если программа запускается неправильно или не подключает элементы, то удалите файлы mcap.dat, *.cmp, *.shp, tool.dat, use.mc11, WFB.bin. Восстановите их из начальной копии.
Файл mcap.dat можно взять из Вашего резерва с настройками. Запустите программу снова. Все должно восстановиться.
   В корневом каталоге программы находятся файлы для исправления отображения русских шрифтов (ru-start.exe, ru-lib.dll, ru-conf.ini) для 32-битной версии программы под Windows7 rus.
   Для этого в файле ru-conf.ini прописывается путь к основному файлу программы MC11.exe (MC11_013xx.exe). Запуск программы делается всегда через файл ru-start.exe.
   Начиная с версии MC11_014xx проблема шрифтов исправлена производителем и данная утилита не нужна.
После запуска программы пропишите необходимые пути к каталогам в настройках  - меню File > Path...> Data и ...> Library.
Именно в этих каталогах программа будет искать файлы и подключенные модули и библиотеки при запуске моделирования и открытии файлов.
Обновления программы заменяют основные исполняемые файлы mc11.exe (mc11xxxx.exe)  и mc11eng.dll.
